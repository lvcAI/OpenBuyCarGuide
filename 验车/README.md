# 验车：别让“新车”变“问题车”

车子到店后，销售会通知你去“验车”。这一步至关重要，一旦签字确认，再发现问题，4S店就可能不认账了。新手别怕麻烦，照着清单一条一条查。

## 1、验车带什么？工具和“军师”都不能少

验车时最好带两样东西：一是“验车清单”（可以在小红书、B站上搜，打印出来照着查），二是“懂车的朋友”。我当时就拉上了星总一起去，他比我细心，帮我发现了不少细节问题。

如果没有懂车的朋友，也可以花钱找“第三方验车师”（抖音上能搜到），花几百块钱买个放心。我当时本来想找验车师，但时间太赶，幸好有星总帮忙。

## 2、验车重点：从“外观”到“车况”，一个都不能漏

验车可以分三步来，新手也能轻松上手：

- **第一步：查外观**：绕车走一圈，看车漆有没有刮花、凹陷，玻璃有没有裂纹。重点看车门边缘、保险杠这些容易被磕碰的地方。可以用手机手电筒照一下，小划痕会更明显。现在车都有很多传感器，也要重点关注。我当时看的车是10月份生产的，很新，车漆没什么问题。
- **第二步：查“三码一致”**：这是避免买到“拼装车”的关键。验车阶段主要核对两个编码：① 车架号（前挡风玻璃左下角）；② 车辆合格证上的车架号。这两个编码必须一致。行驶证需要在后续上牌流程完成后才能拿到，届时可再核对行驶证上的车架号与车辆一致。
- **第三步：查车况**：打开车门，插入钥匙（或通电），看仪表盘有没有故障灯亮起；启动发动机，听声音是不是平顺，有没有异响；试试空调、音响、灯光这些功能好不好用；检查轮胎和车窗的生产日期（比如轮胎上的“4425”代表2025年第44周生产），和车辆生产日期别差太远（一般不超过3个月）。我当时发现车窗是三个批次的，有点担心，但车友群里说“只要不是库存车就没事”，而且车机也是最新版本，也就放心了。

如果验出问题（比如车漆刮花、功能故障），一定要让4S店处理：要么修复，要么换车，要么降价。别听销售说“小问题不影响使用”，坚决维护自己的权益。
