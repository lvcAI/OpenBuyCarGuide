# 贴膜：别为“免费膜”买单

贴膜是提车后的“常规操作”，主要作用是隔热、防紫外线，保护车内隐私。但这里面的“坑”很多，新手一定要注意。

##### 1、免费膜别要：质量差还套路多

4S店常说“送你全车膜”，但这种膜大多是劣质膜，隔热效果差，用几个月就会起泡、掉色。更套路的是：贴免费膜时，销售会说“前挡膜要加钱”“侧后挡膜升级要加钱”，最后还是让你花钱。

我当时提车后，4S店也送了膜，但我试驾时发现“太阳晒得胳膊疼”，知道免费膜没用，就决定自己加钱贴好的。

##### 2、贴膜怎么选？别被“高科技名词”忽悠

贴膜选“前挡+侧后挡”就行，重点看“隔热率”（越高越好，一般50%以上就不错）和“透光率”（前挡透光率要大于70%，不然影响视线）。别信“第五代护肤膜”“航天级材质”这些噱头，选知名品牌（比如龙膜、3M）的中端产品就行，预算控制在2000-3000元。

我当时在4S店合作的贴膜店被推荐“第四代隔热护肤膜”，报价3580元，销售说可以用4S店赠送的基础膜抵扣1000元，初步谈至2280元。后来我在豹旅车友群咨询，老哥们都建议这个预算可以贴龙膜、3M等知名品牌，我又跟销售还价到2000元成交。贴完后隔热效果确实明显，但事后了解到同价位能买到更知名的品牌，还是有些后悔。这里提醒大家，贴膜前一定要在车友群或专业平台查好行情，最好带懂行的朋友同行，避免被套路。
