#### 签合同、付款、买保险：把“钱”的事搞清楚

验车没问题后，就可以签正式的《购车合同》、付款和买保险了。这一步要格外细心，涉及到钱的地方别马虎。

##### 1、签合同：逐字看，别跳页

《购车合同》比订车时的订单更详细，一定要逐字阅读，重点看这几点：

- 车辆信息和价格：和订车订单一致，别出现“低配变高配”“价格上涨”的情况。
- 违约责任：比如“4S店延迟交车怎么赔”“车辆有质量问题怎么处理”。
- 附加条款：之前谈好的赠品、福利，一定要写进合同里，别留空白。

如果销售催你“快点签，都是标准合同”，别听他的——标准合同也可能有“猫腻”，比如藏着“强制在店保养”的条款。我当时就仔仔细细看了合同，确认没问题才签字。 

##### 2、付款：保留好凭证

付款时尽量用银行卡转账，别用现金。转账备注“XX车型购车款”，并保留好转账记录、发票（分为“机动车销售统一发票”和“购置税发票”，新能源车免购置税，只有前者）。这些凭证是后续上牌、维权的关键，别弄丢了。

我当时负了首付，贷款 6.5w，发票也让销售当场开了，妥善收好。
