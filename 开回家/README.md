#### 开回家：新手开车别紧张

所有手续办完，终于可以把车开回家了。新手别紧张，记住这几点：

- **找老司机陪驾**：我是2017年初考的驾照，之后很少开车，属于典型的“本本族”，所以提车时特意拉上了同事肥鹅、波哥（都是有多年驾龄的老司机），还叫上了朋友星总，组成“四人提车小分队”。肥鹅全程帮我开车回家，波哥和星总则帮我留意路况和车辆状态，帮我避开了不少新手容易犯的错，也让我心里踏实很多。
- **熟悉车辆功能**：出发前再熟悉一遍灯光、空调、智驾功能的操作，别在马路上低头找按键。
- **规划好路线**：提前用导航规划好路线，避开拥堵路段。我当时开回家时，百度导航走错了路，多绕了29公里，还好电量够，没被扔在路上。

开回家后，记得找好停车位——我当时提前联系了小区附近的停车场，花300元/月租了个车位，避免“车开回来没地方停”的尴尬。
