# 上牌：新能源车上牌更省心

上牌是买车的“最后一道手续”，新能源车比燃油车更省心，因为不用摇号（部分城市），还能免购置税。上牌可以自己办，也可以让4S店代办（一般收500-1000元服务费），最好让4S店免掉，可以谈。

## 1、上牌前的准备：先申请“新能源指标”

买插混或纯电车，要先在当地的“新能源汽车推广应用管理平台”申请指标。比如我在深圳，就通过“深圳市小汽车增量调控管理信息系统”申请，每个月5号、15号、25号为指标审核时间，审核工作日就能出结果。

如果之前在摇燃油车牌照，要先取消申请，再申请新能源指标。我当时就是因为没及时取消，错过了当月的审核。

## 2、上牌流程：自己办也不难

如果自己上牌，流程大概是这样，有的地方也不需要跑线下：

- ① 提前在“交管12123”APP上预约上牌时间，准备好身份证、购车发票、车辆合格证、交强险保单等资料；
- ② 按预约时间到车管所，先进行车辆查验（工作人员检查车辆外观、车架号等是否与资料一致）；
- ③ 查验通过后到业务大厅提交资料，然后选号（可现场50选1，也可提前在APP上自编或随机选号）；
- ④ 缴纳工本费（车牌100元、行驶证10元等），之后可选择现场领牌（部分车管所支持）或邮寄到家，行驶证和登记证书一般当场就能领取。
