#### 抵押、放款：贷款车主的“必经之路”

如果是贷款买车，付完款后还要办理“车辆抵押”和“放款”手续，这一步一般4S店会帮你代办，但要配合提供资料，比较要查征信。

车辆抵押，像在深圳的话是在深圳交警公众号里面做的抵押预登记，销售会告诉你怎么操作，跟着教程来就行。

抵押的意思是：把车辆的“所有权”暂时抵押给贷款机构（比如弗迪租赁），办理时需要提供身份证、车辆登记证书等资料。这里要明确，行驶证和车辆登记证书上的车主信息都是本人。直到还清贷款，带着身份证、贷款结清证明到车管所办理解押手续，车辆才完全属于我，解押全程免费。

征信没有问题有后，抵押预登记也做了，就等销售人员等放款通知，放款后4S店才会把车交给你（也就是“提车”）。
