# 等车/贷款：耐心等车，别踩“贷款坑”

订完车之后，就是“等车”环节。如果是热门车型，可能要等一两个月；如果有现车，一周左右就能提。这段时间可以同步办理贷款（如果需要的话）。

## 1、等车：别“被动等待”，主动跟进

很多人订车后就天天盼着销售打电话，其实更靠谱的是“主动问”。我当时等豹旅时，每隔几天就给销售发消息，问“车子有没有排产”“什么时候发车”。一方面能了解进度，另一方面也让销售知道你“很在意”，不会把你的订单往后排。

如果等车时间太长（比如超过约定时间），可以找销售协商补偿，比如多送一次保养、一个车载充电器之类的。我当时因为孩子出生，没太多精力催车，但销售还是主动告诉我“车子发出来了”，算是比较负责。

## 2、贷款：选对方案，别被“融资租赁”吓住

如果全款买车，这一步可以跳过；如果贷款，要注意两个关键点：

- **贷款方案**：优先选“厂家金融”，比如比亚迪的“汽金贷款”，一般有免息政策。我当时选的是“两年24期免息”，不用付利息，只需要还本金，压力小很多。别选第三方金融机构的贷款，利息通常很高。
- **融资租赁注意事项**：我当时遇到一个问题——4S店说只能做“弗迪融资租赁”，我在抖音上看到有人说“融资租赁是租车，不是买车”，吓了一跳。后来才搞清楚：弗迪是比亚迪的子公司，这种融资租赁并非以往的“以租代购”，车辆登记证书和行驶证上都会登记我的名字，只是在贷款还款期间，车辆所有权暂时抵押给弗迪租赁，待贷款全部结清后，办理完解押手续，车辆所有权就完全归我了。

如果对贷款方案有疑问，别只听销售的，一定要找“官方客服”（比如弗迪租赁的客服）或“车友群”确认。我当时就是在豹旅车友群里问了老哥们，才放心办理的。
