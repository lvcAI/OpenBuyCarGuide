# 提车

上牌办完后，就可以“提车”了——这是最开心的一步，但也别光顾着激动，还有几件事要做：

##### 1、提车仪式：别被“形式”忽悠

很多4S店会搞“提车仪式”，比如献花、合影、挂红绸带，这些都是“增值服务”，别因为仪式热闹就忘了查车。我提车时，4S店给了一束花，让我和车合影，但车身上还有水渍，车门缝里有小石子，我当场让他们重新洗了车。

质保 别被忽悠了

提车时一定要拿齐“关键凭证”：身份证、购车发票、行驶证（上牌后领取）、登记证书、保险单，少一样都不行。我当时遇到一个销售，差点没给我行驶证，说“要留着办手续”，我坚决要了回来——行驶证是车主的凭证，不能放在4S店。

4S 店的人一个劲的推销他们的延保，其实BYD APP 已经有精诚延保，价格还比他们便宜，别看他给了看似很多内容，其实大多都不会用到。
