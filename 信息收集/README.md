# 二、信息收集
大致需求确定后，接下来就是“扫盲”阶段。对不懂车的人来说，所有品牌看着都一样——哪怕你有心仪的品牌，也别先把自己框死，多看看才能选到最适合的。

## 1、互联网应用：新手的“购车字典”
现在获取车讯的渠道很多，最实用的是专业汽车平台和短视频APP：

懂车帝、汽车之家：这两个是汽车资讯的“天花板”，从新车参数、各地成交价，到车主真实口碑、论坛讨论，一应俱全。新手可以先在上面按“价格”“动力类型”“配置”筛选车型，把符合需求的车列个清单。上面还有“询底价”功能，输入手机号就能收到当地4S店的报价，省得自己跑断腿。

B站、抖音、知乎：这些平台适合“补常识”。B站有很多专业车评人做的深度解析，从车身架构到配置对比，讲得很通俗；抖音上有不少4S店销售发的实车视频，能直观看到车的外观内饰；知乎上的“车主真实体验”帖子，能帮你避开很多4S店不会说的“坑”。比如我当时就是在B站刷到智驾测试视频，才确定一定要买带辅助驾驶的车。

## 2、亲朋好友：参考但不盲从
身边买过车的亲友是“活攻略”，他们的用车体验很真实。比如我姐夫在吉利做智能座舱，我就特意咨询他新能源车的技术趋势；同事波哥买过车，我问他贷款和上牌的流程，省了很多麻烦。

---

⚠️注意：所有人的意见都只能作为参考。每个人的需求不一样，有人看重动力，有人看重空间，别人觉得好的车，不一定适合你。比如我姐觉得轿车操控好，但我需要大空间装东西，最后还是选了旅行版。
