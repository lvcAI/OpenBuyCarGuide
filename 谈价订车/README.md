#### 四、谈价订车：别做“冤大头”，学会“博弈”

看好车之后，就到了最关键的“谈价”环节。新手别不好意思开口，销售都是“看人下菜”，你越犹豫，越容易被坑。问清楚每一个款项的流程和方式。如果有贷款的话，重点注意贷款的方式，走哪一家等等。

##### 1、谈价的核心：先谈“裸车价”，再谈“福利”

很多4S店会把“裸车价”“保险”“服务费”混在一起说，让你搞不清实际花了多少钱。正确的流程是：**先敲定裸车价，再谈赠品和福利**。

- **裸车价**：提前在懂车帝、汽车之家上查当地的“成交价”（不是厂家指导价），心里有个底价。我当时查豹旅旗舰型的成交价在13.5W左右，谈的时候就以此为基础，要求销售“至少降到这个价”。如果销售说“做不了”，可以说“别的店给我报了更低的价”，逼他去申请优惠。我当时就是用“坪洲店能给4000现金优惠”这个条件，让罗湖北的销售也给了同样的优惠，并额外争取到了电池电机护板。
- **福利和赠品**：别要“不值钱”的赠品（比如钥匙扣、脚垫），要实用的。比如可以要“电池电机护板”（保护底盘，走烂路不担心磕碰）、“免费保养”（能省几百块）、“底部装甲”（防生锈）等等。这些东西自己买也要花不少钱，谈价时一定要提，能多要一样是一样。

##### 2、避开“隐形消费”陷阱

销售常用来坑新手的“隐形消费”有几个，一定要提前问清楚：

- **服务费**：比如“贷款服务费”“上牌服务费”，很多4S店会收2000-5000元。其实这些费用是可以谈的，我当时就以“别的店不收服务费”为由，让销售把上牌服务费免了。
- **捆绑销售**：比如“必须在店里买保险”“必须加装饰才能提车”。如果不想在店里买保险，可以说“我自己有渠道”，如果销售不同意，就换一家店。我当时虽然在店里买了保险，但提前在车友群里问了行情，确认价格合理才签的。

##### 3、订车签什么？别只签“定金条”

谈好价格后，就该订车了。这时候一定要签正式的《购车订单》，而不是简单的“定金条”。订单上必须写清楚这几点：

- 车辆信息：车型、配置（比如“豹旅150km旗舰型”）、颜色、生产日期（避免买到库存车，一般三个月内的算新车）。
- 价格明细：裸车价、保险费、服务费、购置税（新能源车免购置税）等，每一项都要写清楚，总落地价别含糊。
- 赠品和福利：电池护板、保养、脚垫这些，写在“附加条款”里，别相信销售的“口头承诺”。
- 提车时间：如果没有现车，要写清楚“预计提车时间”，以及“逾期不提车的赔偿方案”（比如退定金）。

定金和订金要分清：“定金”是不能退的（如果买方违约，定金不退；卖方违约，双倍返还）；“订金”可以退。如果不确定要不要买，尽量签“订金”协议。我当时交了2000元定金，因为订单写清楚了车型和提车时间，后面没出问题。

当这些聊好时，以 BYD 为例会让在 BYD app 下订单，选择对应颜色和版本、经销商店。
