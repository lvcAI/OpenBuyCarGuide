# 一、确定需求
## 1、车是贬值品
当有了买车的念头时，第一个问题就是要问自己：是不是真的需要买一辆车？

首先要明确一点：车是贬值品。对普通人来说，不存在买一辆车去炒作——像刚出来的小米SU7那样引发抢购热潮的“网红车”，不在本文的讨论范围之内。评论区有位老哥晒出自己一千多万的资产说“买国产车就是韭菜”，那他也不是本文的适用对象。

知名“买个der”的博主说得很对：“买车，买个der，等等党永远不亏”，尤其是像比亚迪这些车厂，近年半年一小改、一年一大改，买了就等着被“背刺”。那可不是心疼一点点，都是真金白银啊。比如我的豹旅刚提完车，厂家就申报了新款，升级了电机和电池包，纯电续航直接干到200km。你说，这找谁说理去？

## 2、购车的理由
车在很大程度上是“面子工程”。拥有一辆车，尤其是一辆好车，是人之常情。我也想开好车、开豪车，但能力有限，钱包实在遭不住。十来万上下的购车预算，已经是普通家庭能拿出的“体面”了。

像我，过年要带橙子（我爱人）和刚出生的穗穗回家，都结婚生子了还没添置车，加之家里确实需要一辆车，这就是刚需。也有人是因为过年走亲戚被“催生”买车，或是工作通勤距离太远不得不买。总之，购车理由一定要能说服自己，并且确实能改善生活状况。

评论区有老哥说我买车是冲动消费，觉得十几万的车带不来“人前显贵”的效果。可我只想说：“未经他人苦，莫劝他人善”，这句话放在这里再合适不过——你的需求，只有自己最清楚。

## 3、买什么样的车？
不管买什么，最终都要回归“需求”和“价格”。第一步先别着急考虑预算，认真和自己、家人梳理清楚买车的用途，以及后续能负担的隐形费用（比如保险、油费、停车费等）。

先从车型类型选起，目前主流的车型无非这几种，各自的优缺点很鲜明，新手可以直接对号入座：

- 燃油车：最传统的车型，靠汽油驱动。优点是加油快（几分钟就能加满）、续航长（一箱油跑五六百公里很常见），不用依赖充电桩；缺点是部分城市限牌（比如北京、上海），油价波动大，长期使用成本不低。

- 纯电动车（BEV）：完全靠电池供电驱动。优点是日常使用0油耗，电费比油费便宜很多，保养也简单（不用换机油）；缺点是有“续航焦虑”（冬天续航会缩水），长途出行需要找充电桩，对充电条件要求高。

- 插电式混动（PHEV）：可油可电的“全能选手”，我的豹旅（海豹06 DM-i旅行版）就属于这种。电池容量比纯电车小，但支持外接充电，纯电续航可达150km，日常通勤用电足够；没电了就切换成燃油模式，和普通燃油车没区别，还能上新能源绿牌（部分城市不限行）。

- 增程式（EREV）：本质是电动车，靠电机驱动车轮，但车上装了个“增程器”（可以理解为小型发电机）。这个增程器不直接驱动车轮，只在电池没电时启动发电，相当于给电池“充电”，所以不用担心里程问题，也能上绿牌。

- 油电混动（HEV）：和插混有点像，但不能外接充电，电池靠刹车回收能量充电。优点是比传统燃油车省油，不用依赖充电桩；缺点是不能上绿牌，享受不到新能源政策优惠。

除了动力类型，还要考虑座舱舒适度（比如有没有座椅按摩、大尺寸屏幕）、是否需要辅助驾驶等。把这些需求列出来，目标就清晰了。

我当时就明确了四个核心需求，按优先级排序：
- 1. 长续航（不管是电还是油，跑高速不慌）；
- 2. 有智驾（新手开车怕磕碰，辅助驾驶能帮大忙）；
- 3. 价格在10-15W之间（符合预算）；
- 4. 最好有座椅按摩（长途坐车舒服）。你也可以照着这个方式，把自己的需求写下来。
