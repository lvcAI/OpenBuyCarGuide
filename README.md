# 一个人不懂车的人怎么买车

作者自述：

此前写的豹旅买车记系列，莫名的得到了流量“宠幸”，四篇文章完整翔实的记录了我买车的全过程，也说明这个系列就是许多像我这样普通人所关注的，仅此[豹旅买车记（四）：一个不懂车的人去提车、加钱贴膜](https://mp.weixin.qq.com/s/NleC-VIlHzcFX-lU2iLiyw)一篇截止目前得到 5W+ 传播，其中观点各异，大多集中加钱贴膜的，也有很多读者对车辆本身感兴趣，直呼“学到了”。

如果您是不了解车但要买车跟我一样。虽然，您能够从四篇文章里看到完整购车流程，但是巨大的篇幅和不相干的内容，对于想要快速了解的人来说，还是偏啰嗦了。

因此，我重新整理一篇文章，来帮助所有像我一样窘境的朋友，让您对买车这件事情，不再有心理负担，快速上手。

我的省流版本：**确定需求** → **信息收集** → **看车试驾** → **谈价订车** → **等车/贷款** → **验车** → **签合同**、**付款**、**买保险** → **抵押**、**放款** → **上牌** → **提车**(装护板)→ (**贴膜**)→ **开回家**。

申明：内容基于我个人的知识和体验，有些地方不尽和有错误也难免，本文内容仅作参考。可能和您实际有 出路，基本涵盖了可能出现的情况，也欢迎在评论区指出和分享。

```markmap
# 不懂车买车全流程

## 1️⃣ 确定需求
- 车=贬值品
  - 半年一小改 / 一年一大改
  - 不存在“理财型”买车
- 刚需自检
  - 通勤距离 / 家庭人口 / 面子需求
  - 预算上限+隐形费用（保险/油电/停车）
- 车型速配
  - 燃油 | 纯电BEV | 插混PHEV | 增程EREV | 油混HEV
- 需求清单（示例）
  1. 续航≥X
  2. 智驾
  3. 10-15W
  4. 座椅按摩

## 2️⃣ 信息收集
- 工具
  - 懂车帝/汽车之家（参数+底价）
  - B站/抖音/知乎（评测+避坑）
- 人脉
  - 亲友经验仅作参考
- 输出：3-5款候选车清单

## 3️⃣ 看车试驾
- 渠道
  - 4S店（试驾+现车）
  - 商超店（纯看颜值）
- 看车要点
  - 空间：前/后排+后备箱
  - 配置：智驾/车机/充电口/儿童座椅接口
  - 做工：缝隙/气味
- 试驾重点
  - 动力&刹车
  - 隔音&底盘
  - 功能实操（ACC/自动泊车）
- 带：身份证+驾驶证（≥1年）

## 4️⃣ 谈价订车
- 先谈裸车价（查成交价→压价）
- 再谈实用赠品
  - 电池护板/免费保养/底部装甲
- 避坑
  - 服务费（上牌/贷款）可谈
  - 拒绝捆绑保险/装饰
- 签《购车订单》
  - 车型/颜色/生产日期
  - 价格明细+赠品
  - 提车时间&逾期赔偿
  - 定金vs订金（可退性）

## 5️⃣ 等车/贷款
- 主动跟进排产→发车
- 贷款
  - 优先厂家免息
  - 融资租赁≠以租代购（行驶证写自己）
  - 资料：身份证/合同/合格证

## 6️⃣ 验车
- 带：验车清单+懂车朋友/第三方
- 三步
  1. 外观划痕&玻璃
  2. 三码一致（车身/合格证/行驶证）
  3. 车况：故障灯→功能→轮胎/玻璃日期
- 问题当场换车或降价

## 7️⃣ 签合同·付款·保险
- 合同
  - 逐条读：车辆信息/价格/违约责任/附加条款
- 付款
  - 银行转账+备注，留发票（机动车销售统一发票）
- 保险
  - 交强+车损+三者200W+不计免赔
  - 价比群友，拒绝全险推销

## 8️⃣ 抵押·放款
- 4S代办抵押（资料齐）
- 1-3个工作日放款后放行

## 9️⃣ 上牌
- 新能源：先申指标（地方平台）
- 两种
  - 自办：车管所验车→选号→领牌
  - 代办：500-1000元省心

## 🔟 提车
- 拿齐5件
  - 身份证/发票/行驶证/登记证书/保险单
- 仪式=拍照，重洗+再验外观
- 装护板（电池/电机）现场盯工

## 1️⃣1️⃣ 贴膜
- 拒绝4S免费劣质膜
- 选品牌龙膜/3M中端
  - 前挡透光率≥70%
  - 隔热率≥50%
  - 预算2000-3000元

## 1️⃣2️⃣ 开回家
- 老司机陪驾
- 再熟悉灯光/智驾/空调
- 导航避拥堵，提前租车位
```
